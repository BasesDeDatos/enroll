Usage

<div class="row">
	<div class="equal_height xs">
		<div class="col-xs-6">
			<div class="valign-middle">Content</div>
		</div>
		<div class="col-xs-6">
			<div class="valign-middle">Content</div>
		</div>
	</div>
</div>

The div tag with class equal_height must also have one of the following classesxs, sm, md or lg. These classes define the breakpoint from which the vertical alignment should start.

You must also add a tag that has a valign-* class just inside yourcol-* tag to define the property of the vertical alignment. Available classes are valign-top, valign-middle and valign-bottom.
